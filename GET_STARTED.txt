====================================================================
⚡ QUICK START (3 STEPS)
====================================================================

STEP 1: Install Dependencies
--------------------------------------------------------------------
Open terminal in this folder and run:
    npm install

STEP 2: Start Development Server
--------------------------------------------------------------------
    npm run dev

Your portfolio will open at http://localhost:3000

STEP 3: Customize Your Info
--------------------------------------------------------------------
Update these files with your personal information:
    → src/components/Contact.jsx (email, LinkedIn, GitHub)
    → src/components/Footer.jsx (social links)
    → src/components/Projects.jsx (project links)

====================================================================
🚀 DEPLOY TO GITHUB PAGES
====================================================================

1. Update vite.config.js - Change line 7:
   base: '/your-repo-name/',

2. Push to GitHub:
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/USERNAME/REPO.git
   git push -u origin main

3. Enable GitHub Pages:
   → Go to repository Settings → Pages
   → Source: Select "GitHub Actions"
   → Wait 2-3 minutes

4. Your site will be live at:
   https://USERNAME.github.io/REPO/
